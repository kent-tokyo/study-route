本サイトについて

本サイト（study-route）は、数学・哲学・AWS・コンピュータサイエンス・化学・会計など複数分野の概念学習を支援する無料のWebアプリケーションです。学習コンテンツはAI（Anthropic社のClaude）により生成され、正確性を確認しています。

免責事項

本サイトのコンテンツはAIにより生成されたものであり、教育目的でのみ提供されます。正確性には努めていますが、情報の正確性・完全性・最新性を保証するものではありません。重要な情報は権威ある情報源でご確認ください。

特に会計コンテンツについて：説明は日本基準（J-GAAP）を基本とし、必要に応じてIFRSとの差異に言及しています。本コンテンツは専門的な会計・税務アドバイスを構成するものではありません。

プライバシー

本サイトは個人情報を収集しません。学習進捗や設定はすべてブラウザのlocalStorageに保存され、サーバーに送信されることはありません。

使用技術

  • Next.js (Static Export) + React + TypeScript
  • Hosted on GitHub Pages
  • Content generated by Claude (Anthropic) and Google Gemini (illustrations)
  • KaTeX for mathematical notation
